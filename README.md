# AI-Review-Frontend

## Installation

This project uses Yarn as the package manager. To install dependencies:

```bash
yarn install
```

## Development

To start the development server:

```bash
yarn start
```

To run tests:

```bash
yarn test
```

To build the project:

```bash
yarn build
```

To lint the code:

```bash
yarn lint
```
